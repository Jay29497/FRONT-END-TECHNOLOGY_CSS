CSS Pseudo-elements..!

CSS Pseudo-elements allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS3 intorduced a new double-colon (::) syntax for pseudo-elements to distingush between them and pseudo-classes.

Selector::pseudo-element { property: value; }
